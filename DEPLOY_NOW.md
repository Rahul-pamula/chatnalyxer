# 🚀 Deploy to Render - Step by Step

## ✅ What You Have Ready
- Working backend with Supabase

## 📋 Deployment Steps (30 minutes)

### Step 1: Prepare Code for Deployment (5 min)

```bash
# 1. Add Gemini API key to .env (already done!)
# 2. Make sure .env is in .gitignore
echo ".env" >> .gitignore

# 3. Commit all changes
git add .
git commit -m "Prepare for Render deployment with Gemini AI"

# 4. Push to GitHub
git push origin dev_otp_flow
```

---

### Step 2: Deploy Backend to Render (10 min)

1. **Go to Render**: https://render.com
2. **Sign up** with GitHub
3. **New Web Service** → Select your `chatnalyxer` repository
4. **Configure**:
   ```
   Name: chatnalyxer-backend
   Region: Singapore
   Branch: dev_otp_flow
   Root Directory: chatnalyxer-backend
   Build Command: pip install -r requirements.txt
   Start Command: uvicorn app.main:app --host 0.0.0.0 --port $PORT
   ```

5. **Environment Variables** (click "Environment"):
   ```
   DATABASE_URL = <your-supabase-url>
   GEMINI_API_KEY = AIzaSyAewHn3G08zoHfuwVxrNMeFl3m2O-MiI3M
   SECRET_KEY = <auto-generated>
   ALGORITHM = HS256
   ACCESS_TOKEN_EXPIRE_MINUTES = 30
   OTP_EXPIRY_MINUTES = 5
   OTP_MAX_ATTEMPTS = 3
   ```

6. **Create Web Service** → Wait 5-10 minutes

7. **Note your URL**: `https://chatnalyxer-backend.onrender.com`

---

### Step 3: Deploy WhatsApp Service (10 min)

1. **New Web Service** → Same repository
2. **Configure**:
   ```
   Name: chatnalyxer-whatsapp
   Region: Singapore
   Branch: dev_otp_flow
   Root Directory: whatsapp-integration
   Build Command: npm install
   Start Command: node otp-service.js
   ```

3. **Environment Variables**:
   ```
   PORT = 3001
   ```

4. **Create Web Service** → Wait 5-10 minutes

5. **Note your URL**: `https://chatnalyxer-whatsapp.onrender.com`

6. **Update Backend**:
   - Go back to backend service
   - Add env var: `OTP_SERVICE_URL = https://chatnalyxer-whatsapp.onrender.com`
   - Backend will auto-redeploy

---

### Step 4: Update Mobile App (5 min)

1. **Edit** `chatnalyxer-mobile/src/config.ts`:
   ```typescript
   export const BASE_URL = "https://chatnalyxer-backend.onrender.com";
   ```

2. **Commit and push**:
   ```bash
   git add chatnalyxer-mobile/src/config.ts
   git commit -m "Update mobile app to use deployed backend"
   git push
   ```

---

### Step 5: Test Deployment

```bash
# Test backend
curl https://chatnalyxer-backend.onrender.com/health

# Test WhatsApp service
curl https://chatnalyxer-whatsapp.onrender.com/health

# Test mobile app with Expo
cd chatnalyxer-mobile
npx expo start
```
