# Video Recording Guide - Quick Start

## 🎥 Recording Steps (Mac)

### Option 1: QuickTime (Recommended)
1. Open **QuickTime Player**
2. File → New Screen Recording
3. Click record button
4. Select area or full screen
5. Start your demo!

### Option 2: Built-in Screenshot Tool
1. Press **Cmd + Shift + 5**
2. Select "Record Entire Screen" or "Record Selected Portion"
3. Click Record

## 📱 What to Record

### Setup (Before Recording)
1. **Clean your WhatsApp test group**
   - Remove old messages
   - Prepare demo messages to send

2. **Test your app**
   - Ensure WhatsApp is connected
   - Verify AI extraction works
   - Test notifications
   - Test AI chat

3. **Close unnecessary apps**
   - Hide personal info
   - Clean desktop
   - Turn off notifications

### Demo Flow (2 minutes of recording)

#### Scene 1: Setup (15 seconds)
- Show app home screen
- Show WhatsApp connection status

#### Scene 2: Message Arrives (30 seconds)
- Send WhatsApp message: "Assignment submission deadline is tomorrow at 5 PM"
- Show message in WhatsApp
- Switch to app
- Show AI extracted the deadline

#### Scene 3: Calendar & Notification (30 seconds)
- Open calendar view
- Show event created automatically
- Show notification received

#### Scene 4: AI Chat (30 seconds)
- Open AI chat
- Ask: "What's due this week?"
- Show AI response with deadline info

#### Scene 5: Dashboard Overview (15 seconds)
- Show main dashboard
- Highlight key features

- **Separate**: Record video first, add voice later (more polished)

## ✂️ Editing (iMovie - Mac)

### Quick Edit Steps
1. Open iMovie
2. Create New Project → Movie
3. Import your recording
4. Trim unnecessary parts
5. Add title cards:
   - Opening: "Chatnalyxer - AI Academic Assistant"
   - Closing: "Never Miss a Deadline Again"
6. Export → File → 1080p

### Title Cards to Add
- **Opening**: Project name + tagline
- **Problem**: "Students miss 40% of deadlines"
- **Solution**: "AI-Powered WhatsApp Analysis"
- **Closing**: GitHub link + contact

## 📤 Upload to YouTube

1. Go to youtube.com
2. Click Create → Upload Video
3. Select your video file
4. Set visibility: **Unlisted** (important!)
5. Title: "Chatnalyxer - Imagine Cup 2026 Demo"
6. Description: Copy from IMAGINE_CUP_SUBMISSION.md
7. Click Publish
8. Copy the link for submission

## ⏱️ Time Estimate
- Recording: 20-30 minutes (multiple takes)
- Editing: 15-20 minutes
- Upload: 5-10 minutes
- **Total**: ~60 minutes

## 🚨 Common Mistakes to Avoid
- ❌ Video too long (over 3 minutes)
- ❌ Poor audio quality
- ❌ Showing personal information
- ❌ App crashes during demo
- ❌ Forgetting to set YouTube to "Unlisted"

## ✅ Quality Checklist
- [ ] Video is under 3 minutes
- [ ] Audio is clear
- [ ] Demo shows all key features
- [ ] No personal info visible
- [ ] YouTube link works
- [ ] Video is "Unlisted" not "Private"
